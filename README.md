# Calorie Checker

Calorie Checker enables you to know how much calorie you should take in a day and  
get recommendation of your next meal.

Ruby version : 2.3.1  
Rails : 5.1.4

## How to start this app

1. Clone repository

2. Install Gems

```
$ bundle install --without production
```

3. Migrate Database

```
$ rails db:migrate
```

3. Run test

```
$ rails test
```

4. Start server

```
$ rails server
```
